CSS Positioning Tutorial

Static Positioning (Default)

Elements with position: static follow the normal document flow. This is the default positioning behavior.

Static Position

Relative Positioning

Elements with position: relative can be moved relative to their normal position using top, right, bottom, and left properties.

Absolute Positioning

Elements with position: absolute are positioned relative to their nearest positioned ancestor.
