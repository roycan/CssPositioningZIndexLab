CSS Positioning Tutorial

Static Positioning

The default positioning. Elements appear in the normal document flow. position: static;

Static

Relative Positioning

Positioned relative to its normal position. position: relative; top: 40px; left: 40px;

Relative

Absolute Positioning

Positioned relative to its nearest positioned ancestor. position: absolute; bottom: 20px; right: 20px;

Absolute

Fixed Positioning

Positioned relative to the viewport - stays in place even when scrolling. position: fixed; top: 20px; right: 20px;

Fixed
The fixed box stays in the top-right corner of the viewport

Sticky Positioning

Toggles between relative and fixed positioning based on scroll position. position: sticky; top: 0;
